# L Cafe — Find Your Summer Pause

نسخه آماده انتشار روی `GitHub Pages` با بازطراحی کامل **کل سایت** بر پایه‌ی `PNG Cutout` آیتم‌ها.

در این نسخه، عکس‌های اصلی منو دیگر به‌صورت تصویر کامل و بک‌گراند خشک استفاده نشده‌اند. هر ۸ نوشیدنی به‌صورت محصول جداشده با پس‌زمینه شفاف در تمام بخش‌های سایت استفاده شده‌اند:

- صفحه شروع
- هر دو مرحله کوییز
- صفحه Reveal
- صفحه نتیجه
- پیشنهاد ثانویه
- صفحه «همه نوشیدنی‌ها»

منطق کوییز، متن‌ها، قیمت‌ها، نتایج اصلی و پیشنهاد ثانویه تصادفی حفظ شده‌اند.

## انتشار روی GitHub Pages

محتویات این پوشه را مستقیماً در ریشه مخزن قرار دهید؛ فایل `index.html` باید در ریشه دیده شود.

در GitHub:

1. وارد `Settings` شوید.
2. بخش `Pages` را باز کنید.
3. `Deploy from a branch` را انتخاب کنید.
4. شاخه `main` و پوشه `/(root)` را انتخاب کنید.
5. `Save` را بزنید.

فایل `.nojekyll` برای انتشار مستقیم داخل پروژه قرار دارد.

## اجرای محلی

```bash
python -m http.server 8080
```

سپس باز کنید:

```text
http://localhost:8080
```

## ساختار پروژه

- `index.html` — ساختار صفحات
- `assets/css/styles.css` — طراحی، واکنش‌گرایی و Motion
- `assets/js/data.js` — محصولات، قیمت‌ها، تصاویر و نگاشت کوییز
- `assets/js/app.js` — منطق کوییز، نتیجه و منوی کامل
- `assets/images/` — ۸ تصویر WebP شفاف‌شده‌ی محصولات
- `assets/brand/lcafe-logo.svg` — لوگوی رسمی
- `QA_REPORT.md` — گزارش کنترل کیفیت

## منطق کوییز

- سؤال اول لاین را مشخص می‌کند.
- سؤال دوم نتیجه اصلی را تعیین می‌کند.
- هر ۸ نتیجه اصلی ثابت و حفظ شده‌اند.
- پیشنهاد ثانویه عمداً تصادفی است و با نتیجه اصلی یکسان نمی‌شود.

## فونت‌ها

- فارسی: `Vazirmatn`
